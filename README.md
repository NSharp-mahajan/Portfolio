## 🚀 Quick Start

### **Prerequisites**
- Node.js 18.0+ 
- npm or yarn package manager

### **Installation**
```bash
# Clone the repository
git clone https://github.com/NSharp-mahajan/Portfolio.git

# Navigate to project
cd Portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

### **Development**
```bash
# Development server with HMR
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🎨 Design Philosophy

### **Performance First**
- **Lazy Loading**: Images load on-demand
- **Optimized Animations**: CSS transforms for 60fps
- **Minimal Bundle**: Tree-shaking and code splitting
- **Hardware Acceleration**: GPU-accelerated animations

### **Accessibility**
- **Semantic HTML**: Proper heading hierarchy and ARIA labels
- **Keyboard Navigation**: Full keyboard accessibility
- **Screen Reader Support**: Descriptive alt text and roles
- **Focus Management**: Logical tab order and focus indicators

### **User Experience**
- **Progressive Enhancement**: Works without JavaScript
- **Smooth Transitions**: Micro-interactions throughout
- **Loading States**: Visual feedback for all operations
- **Error Handling**: Graceful degradation
